# 🎬 Movie Ticket Booking Backend API

## 📌 Project Overview

This project demonstrates the development of a **backend system for a Movie Ticket Booking platform**. The application provides RESTful APIs to manage users, movies, theatres, show schedules, and ticket reservations. It focuses on building a scalable server-side architecture that handles authentication, data management, and booking workflows efficiently.

---

## 🎯 Project Objectives

Develop a backend API for managing movie ticket booking operations.

Provide secure user authentication and authorization.

Manage movies, theatres, show timings, and ticket reservations.

Ensure smooth communication between the frontend and database.

---

## 🗂 Project Features

Feature | Description
User Authentication | Register and login functionality with secure authentication
Movie Management | Add, update, and view movie details
Show Scheduling | Manage show timings for different movies and theatres
Seat Booking | Allow users to reserve seats for available shows
REST API | Provides structured API endpoints for frontend integration

---

## 🛠 Tools & Technologies

Backend: Node.js, Express.js

Database: MongoDB / Mongoose

API Type: RESTful API

Development Tools: VS Code, Git, GitHub

Approach: API development, database management, authentication, and booking workflow handling
